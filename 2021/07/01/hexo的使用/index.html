Hexo

安装前提

安装hexo需要先安装以下两应用程序：

  • Git
  • Node.js

安装git

git的大名，想必程序员都知道，作为目前世界上数一数二的分布式版本控制系统，安装教程也不少

  • windows : 想安装git直接去git官网，简单点的话教程一路next，注意一下安装地址什么的就差不多了。

  • Mac : 可以通过Homebrew，Xcode等软件安装

  • Linux : 注意区分一下Linux的版本

安装Node.js

hexo博客基于Node.js编写，所以需要安装这个工具

  • 安装包：前往 node.js官网在LTS中下载对应的版本安装包就行了！
  • 使用nvm或者nvs，因为像Mac和Linux这样的命令安装也有可能版本较老，而且三种常见的操作系统都可以用这种方法。

安装完成之后可以在命令控制符（推荐使用git bash）中使用node -v和npm -v查看node和npm的版本号。

创建GitHub个人仓库

GitHub的账号应该也有吧，没有的话注册一个，这个教程网上也很多，邮箱的话最好使用国内的，国外的有些连接不好，然后登录，点击New repository创建一个新的仓库，仓库名需要注意：使用**<你的 GitHub 用户名>.github.io**作为仓库名，否则后面会报错的，

博客编写

首先我们要创建一篇新文章，在本地电脑中运行hexo new [layout]<title>

ps: 【layout】也可以不添加哦！这是表示你放文件的位置的，默认是post

布局 路径
post source/_posts
page source
draft source/_drafts

之后再本地找到对应文件夹下的文章，进行内容编写，这里推荐Typora，非常方便，文章写完了我们可以看在本地运行，就可以在本地上观看样式了，如果OK就可以提交到GitHub上，之后在浏览器中输入

https://github.com//.github.io就可以在线上看自己的博客文章了；
