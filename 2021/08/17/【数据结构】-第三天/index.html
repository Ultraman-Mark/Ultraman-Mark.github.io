leetcode 350题 两个数组的交集II

给定两个数组，编写一个函数来计算它们的交集。

示例 1：

输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2,2]

示例 2:

输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[4,9]

作答思路：

一、hashmap解决

先用一个map存储一个数组中的所有元素，之后再遍历第二个数组，并与每个元素进行比较，存在的就放入list中（因为我们无法实现知道并集的数量，所以不能直接使用用int数组），使用map是考虑了元素重复的问题，可以存储带map的value中，num1每重复一个就使对应的value+1，num2每重复一个就使value-1；

二、双指针

思路是先排序，之后使用双指针从小到大进行遍历，

  • num1[p1] == num2[p2] 就填入list中；
  • num1[p1] < num2[p2] 就使p1++;
  • num1[p1] > num2[p2] 就使p2++;

leetcode 121题 买卖股票的最佳时机

给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。

你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。

示例 1：

输入：[7,1,5,3,6,4]
输出：5
解释：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

示例 2：

输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下, 没有交易完成, 所以最大利润为 0。

作答思路：

使用两个数值，max，min

  • max-记录前面所有数的最小值与当前数的差值（差值小于0应当取0）
  • min-记录前面所有数中的最小值
