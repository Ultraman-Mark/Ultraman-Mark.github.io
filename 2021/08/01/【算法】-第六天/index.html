1.leetcode 3题 无重复字符的最长子串

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入 : s = “abcabcbb”
输出 : 3
解释 : 因为无重复字符的最长子串是 “abc”，所以其长度为 3。

示例 2:

输入 : s = “bbbbb”
输出 : 1
解释 : 因为无重复字符的最长子串是 “b”，所以其长度为 1。

示例 3:

输入 : s = “pwwkew”
输出 : 3
解释 : 因为无重复字符的最长子串是 “wke”，所以其长度为 3。请注意，你的答案必须是 子串 的长度，”pwke” 是一个子序列，不是子串。

示例 4:

输入 : s = “”
输出 : 0

作答思路：

本题属于使用滑动窗口，滑动窗口有点像双指针，`顾名思义，就是有一个大小可变的窗口，左右两端方向一致的向前滑动（右端固定，左端滑动；左端固定，右端滑动）。

双指针：

  • 计算过程和两端点相关的称为双指针
  • 不固定大小
  • 双指针是解决问题的一种方法
  • 双指针可以同向移动可以双向移动
  • 同向移动的双指针和滑动窗口没有任何联系

滑动窗口：

  • 计算过程与两端点表示的区间相关的称为滑动窗口。
  • 默认固定大小的窗口，在一些条件触发的情况下，可能会将其大小进行修改。
  • 滑动窗口本身并不是解决问题的一种方法（或者说算法），它其实就是问题本身。
  • 滑动窗口一定是同向移动的。
  • 滑动窗口是一类问题，不同的问题需要使用不同的算法和数据结构来解决。

以上都是网上抄来的，但我觉得大概就是这个样子的吧，这题的思路是这样的，我们从字符串的每一个字母开始向后搜索，一旦遇到一个和我们窗口内一样的字母，就退出，并将长度与max进行比较，如果大于max，则max=长度。然后比较关键的一点是，如何存储我们窗口中的字母，因为题目中要求是英文字母、数字、符号和空格组成，所以我们可以用hashmap或者hashset这样的类型来存储。

2.leetcode 567题 字符串的排列

给你两个字符串 s1 和 s2 ，写一个函数来判断 s2 是否包含 s1 的排列。

换句话说，s1 的排列之一是 s2 的 子串 。

示例 1：

输入 ：s1 = “ab” s2 = “eidbaooo”
输出 ：true
解释 ：s2 包含 s1 的排列之一 (“ba”).

示例 2：

输入 ：s1= “ab” s2 = “eidboaoo”
输出 ：false

作答思路：

s2中是否包含s1的全排列，比较全排列只要看看字母是否一样，且对应的字母数量是否相等就可以了，因此这题可以用双指针也可以用滑动窗口，我们用滑动窗口，我们从s2的每个字母进行s1大小的窗口进行比较，就能获取对应字符串。
